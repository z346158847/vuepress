# MobaXterm介绍
MobaXterm是一款增强型终端、X 服务器和 Unix 命令集(GNU/ Cygwin)工具箱。
简单地说，把ssh，sftp集成到了一起，界面好看，好用
重点:存储会话有个数限制，英文
